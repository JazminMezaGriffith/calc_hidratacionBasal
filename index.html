¿Cómo se calcula?

Utilizando el método de Holliday-Segar:

  • De 0kg a 10kg, se calcula 100cc por cada kilo.
  • Se suman 50cc por cada kilo de peso por arriba de 10kg, hasta 20kg.
  • De 20kg para arriba, se suman 20cc por cada kilo adicional.

Utilizando el método de Superficie corporal:

  • Se utiliza la fórmula: SC = ( (peso * 4) + 7) / (peso + 90)
  • El resultado se multiplica por 1500 o por 2000.
